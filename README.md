# Nauta Email Categorization System

Sistema de categorización inteligente de emails para Nauta.

## Estructura
- `docs/` - Documentación e instrucciones
- `scripts/` - Scripts Python de categorización
- `data/` - Emails de prueba (CSV)
- `output/` - Resultados JSON
- `tests/` - Pruebas unitarias
